# Voo Demo

This is a blank static page intended for a quick demo deployment on GitHub Pages.

## Deploy (GitHub Pages)
1. Create a new GitHub repo (private or public).
2. Push this folder to the repo's `main` branch.
3. In GitHub: Settings -> Pages -> Build and deployment
4. Source: `Deploy from a branch`
5. Branch: `main` and folder `/ (root)`
6. Save, then wait for the page URL to appear.

## Local preview
Open `index.html` directly in your browser for now.
